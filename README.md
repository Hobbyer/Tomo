## 🧩 프로젝트 소개 | プロジェクト紹介

TOMO(トモ)는 일본어 '友達(ともだち)'에서 영감을 받아  
‘친구처럼 편하고 자연스러운 협업’을 목표로 만든 **실시간 협업 툴**입니다.

TOMO（トモ）は日本語の「友達（ともだち）」からインスピレーションを受けて、  
まるで友達と話すように、自然で楽しいコラボレーションを目指しています ✨

<br>

## 🛠 주요 기능 | 主な機能

💬 **실시간 블록 에디터 | リアルタイムブロックエディタ**  
→ 여러 사용자가 동시에 문서를 작성하고 수정 가능  
→ 複数のユーザーが同時にドキュメントを作成・編集できます。

<br>

🧱 **블록 단위 콘텐츠 구성 | ブロック単位のコンテンツ構成**  
→ Heading, Paragraph, List 등 다양한 요소 지원  
→ 見出し・段落・リストなど、さまざまな要素に対応しています。

<br>

⚡️ **WebSocket 기반 실시간 동기화 | WebSocketによるリアルタイム同期**  
→ 모든 편집 내용이 실시간으로 반영  
→ 編集内容がリアルタイムで反映されます。

<br>

📦 **CRDT 기반 충돌 방지 | CRDTによる衝突防止**  
→ 동시에 작성해도 안전하게 병합  
→ 複数人の同時編集でも、安全にマージできるよう設計されています。

<br>

🌐 **오프라인 모드 (검토 중) | オフラインモード（検討中）**  
→ 인터넷 없이도 임시 저장 후 재접속 시 자동 반영 기능을 고려 중입니다.  
→ オフライン中の作業を一時保存し、再接続後に自動的に反映する機能を検討中です。

<br>
